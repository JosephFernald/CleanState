# CleanState

**Deterministic FSM Framework for Unity and .NET**

A performance-tuned, event-driven state orchestration engine with full debugging and recovery support.

## Get Started

- [Introduction](articles/intro.md) — Learn what CleanState is and why it exists
- [Getting Started](articles/getting-started.md) — Build and run your first state machine
- [API Reference](api/index.md) — Full API documentation

## Key Features

- **Run-until-blocked execution** — Sleeps when idle, zero cost
- **Explicit step pipelines** — Every step has a name, type, and source location
- **Full transition provenance** — Reason, trigger, source, destination, timestamp
- **Pure C# core** — No MonoBehaviour, no coroutines, no Unity types
- **Checkpoint-based recovery** — Restore from domain truth, not FSM position
- **Visual debugger** — Live state, timeline, breakpoints
